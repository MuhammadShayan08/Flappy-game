# Spoken English Pro Quest

A Streamlit spoken-English study game with:

- XP system
- Levels
- Badges
- Speaking missions
- Daily challenges
- IELTS mode
- Leaderboard
- Sound effects
- Animations

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Cloud

Upload these files to GitHub:

- app.py
- requirements.txt

Then connect the repo with Streamlit Community Cloud.
